## 애플리케이션 개요
- 애플리케이션 명칭 : **MobileDeviceInfo**
- 애플리케이션 유형 : **PHP 웹 애플리케이션**
- 기술 구성 : **FastCGI 방식의 Non-Thread Safe PHP 8**
- 데이터베이스 : **MySQL 8**
- 서버 구성 : **Nginx 리버스 프록시 + Apache 기반 백엔드**
